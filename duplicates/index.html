Not familiar with JavaScript? Here are the basics:

  • Declare a local variable: var name; or var name = value;
  • Declare an empty array: var a = [];
  • Make a copy of an array: var b = a.slice();
  • Push elements onto the array: a.push( value )
  • Access a particular element: a[i]
  • Sort the array by string order: a.sort()
  • Iterate over the elements in the array: for ( i in a ) { v = a[i] ... }
  • Declare an empty object / hash-map: var h = {};
  • Add a new key/value pair: h.key = "value" or h["key"] = "value"
  • Access the value stored for a particluar key: h.key or h["key"]
  • Test if the key exists already: h.hasOwnProperty( "key" )
  • Iterate over the keys: for ( k in h ) { v = h[k] ... }
  • Get the length of a string: var s = "foo"; s.length == 3
  • Access the nth character in a string: var s = "foo"; s.charAt(0) == "f"
  • Find the first instance of character/substring: var s = "foo"; s.indexOf('o') == 1; s.indexOf('x') == -1
  • Replace all instances of character X with character Y: s.replace(/X/g, 'Y' )
  • Convert a string to an array: var s = "foo"; var a = s.split(''); // [ 'f', 'o', 'o' ]
  • Concatenate the elements of an array into a string: a.join('')
  • Increment/decrement a variable: x += 1 or x -= 1
